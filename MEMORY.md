## 2. Запуск та Автентифікація (TASK-7)
*   **Веб-сервер**: Запускається вручну через `./kbg.sh serve --port <PORT>` (або з флагом `--debug` у режимі розробки).
*   **Пріоритет перевірки пароля**:
    1. `KBG_WEB_PASSWORD` (змінна оточення) — якщо задана, **завжди** перевизначає та ігнорує будь-які збережені файли облікових даних.
    2. `web_credentials.json` — якщо змінна оточення відсутня, зчитується збережений хеш.
    3. Автогенерація випадкового токену (якщо файл відсутній) — записується у `web_credentials.json` та виводиться у консоль при старті.
