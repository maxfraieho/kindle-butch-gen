# Сторонні компоненти та ліцензії (NOTICE)

Vydra використовує такі сторонні компоненти. Дякуємо їхнім авторам.

| Компонент | Ліцензія | Використання |
|---|---|---|
| [llama.cpp](https://github.com/ggml-org/llama.cpp) | MIT | локальний inference LLM |
| [Hy-MT2-7B](https://huggingface.co/tencent/Hy-MT2-7B) (Tencent) | Apache-2.0 | модель перекладу |
| [Google Gemma 3 4B](https://ai.google.dev/gemma) | [Gemma Terms of Use](https://ai.google.dev/gemma/terms) — користувач приймає умови (включно з [Prohibited Use Policy](https://ai.google.dev/gemma/prohibited_use_policy)) під час встановлення | преміум vision/NER |
| [Tesseract OCR](https://github.com/tesseract-ocr/tesseract) | Apache-2.0 | розпізнавання тексту |
| [comic-text-detector](https://github.com/dmMaze/comic-text-detector) | GPL-3.0 | детекція тексту манґи (встановлюється користувачем окремо, pip) |
| [Calibre](https://calibre-ebook.com/) | GPL-3.0 | конвертація ebook (викликається як зовнішня програма) |
| [mapaki](https://github.com/tsopeh/mapaki) | MIT | пакування манґи для Kindle |
| [whisper.cpp](https://github.com/ggml-org/whisper.cpp) | MIT | (заплановано) перевірка аудіо |
| [StyleTTS2](https://github.com/yl4579/StyleTTS2) | MIT (код) | озвучення |
| Supertonic TTS (ваги) | ліцензія ваг уточнюється — до підтвердження функції на її основі не входять до платних | озвучення |
| [Liberation Fonts](https://github.com/liberationfonts/liberation-fonts) | SIL OFL 1.1 | шрифти typeset |
| [ipa-uk](https://github.com/lang-uk) (vendored) | див. common/vendor/ipa_uk | фонетика наголосів |

Повні тексти ліцензій — у репозиторіях відповідних проєктів.
