# 📱 Просте розгортання kindle-butch-gen на OnePlus 13 (Termux)

Ця інструкція створена для користувачів з **мінімальним досвідом роботи в консолі**. Дотримуйтесь кроків нижче, щоб автоматично налаштувати систему перекладу та синтезу аудіокниг на вашому смартфоні.

---

## 🛠️ Крок 1. Підготовка телефону та встановлення Termux

> [!IMPORTANT]
> Не встановлюйте Termux з Google Play Store (там застаріла версія). Встановлюйте лише з F-Droid.

1. Завантажте та встановіть додаток **Termux** з офіційного сайту [F-Droid](https://f-droid.org/packages/com.termux/).
2. Запустіть Termux на телефоні.
3. Дозвольте доступ до пам'яті телефону. Для цього введіть у консолі команду та натисніть **Enter**:
   ```bash
   termux-setup-storage
   ```
   *(На телефоні з'явиться запит — підтвердьте доступ до файлів).*

---

## 🚀 Крок 2. Запуск автоматичного встановлення

> [!IMPORTANT]
> **Перед запуском команди зверніть увагу:**
> 1. Скрипт проведе повну автоматичну настройку: оновить пакети Termux, встановить Ubuntu-контейнер через `proot-distro`, налаштує OpenCL для прискорення GPU Adreno 830, скомпілює сервіс `llama.cpp` та встановить усі Python-залежності для перекладу та синтезу мовлення (включаючи `stress-uk` та `manga-ocr`).
> 2. **Вимоги до пам'яті**: Для встановлення оточення та моделей вам знадобиться **щонайменше 10-15 ГБ вільного місця** на телефоні.
> 3. **Вимоги до мережі**: Буде завантажено велику модель перекладу (~4.4 ГБ) та модель синтезу мовлення (~128 МБ). Будь ласка, підключіть телефон до стабільного Wi-Fi.

Скопіюйте цю команду, вставте її в Termux і натисніть **Enter**:

```bash
curl -O https://raw.githubusercontent.com/maxfraieho/kindle-butch-gen/master/deploy_oneplus13.sh && chmod +x deploy_oneplus13.sh && ./deploy_oneplus13.sh
```

## 💬 Інтерактивні запити під час встановлення

Під час роботи скрипт звернеться до вас із запитами. Вони створені для максимальної зручності:

### 1. Запит про автозапуск сервісів (Autostart)
Екран покаже запит:
```text
Do you want to configure automatic startup of services (sshd, llama-server, web server) on Termux launch? (y/N):
```
*   **Що ввести**: Натисніть **`y`** та **Enter**, якщо хочете, щоб при кожному запуску Termux автоматично запускався веб-сервер перекладу та SSH-доступ. Це рекомендується, щоб вам не доводилося запускати сервіси вручну після перезавантаження телефону.
*   **Якщо натиснути `n`**: Автозапуск буде пропущено.

### 2. Запит про завантаження моделі перекладу (GGUF Model)
Екран покаже запит:
```text
Translation model Hy-MT2-7B-Q4_K_M.gguf (4.4GB) is missing.
Please choose an option:
  1) Download the default model from Hugging Face (~4.4GB)
  2) Paste a custom download link
  3) Skip downloading for now
Enter choice [1-3]:
```
*   **Варіант `1` (Рекомендовано)**: Скрипт завантажить офіційну оптимізовану модель перекладу з Hugging Face і автоматично перевірить цілісність файлу після завершення завантаження.
*   **Варіант `2`**: Якщо ви хочете вставити власне дзеркало для скачування, оберіть `2` і скрипт чекатиме, поки ви вставите пряме посилання в діалоговому режимі.
*   **Варіант `3`**: Пропустити скачування моделі зараз.

### 3. Запит про завантаження моделі синтезу мовлення (Supertonic 3 Model)
Скрипт перевірить наявність та цілісність моделі TTS і за потреби запропонує її завантажити:
```text
Supertonic 3 TTS model directory is missing or incomplete.
Do you want to download and extract Supertonic 3 TTS model? (Y/n):
```
*   **Що ввести**: Натисніть **`y`** (або просто Enter), і скрипт скачає (~128 МБ) та розпакує модель у правильну робочу директорію.

## 🌐 Крок 3. Використання системи

Після завершення встановлення:

1. **Якщо ви увімкнули автозапуск**: 
   * Просто запустіть додаток Termux на телефоні. Усі сервіси піднімуться у фоновому режимі автоматично.
2. **Перехід у веб-інтерфейс**:
   * Відкрийте будь-який браузер на телефоні або комп'ютері в тій самій Wi-Fi мережі.
   * Перейдіть за адресою: **`http://localhost:5000`** (на самому телефоні) або за IP-адресою вашого OnePlus 13 (наприклад, `http://192.168.3.196:5000`).
3. **Керування перекладом**:
   * Ви можете керувати та дивитися статус перекладу книжок та манги через дашборд. Усі сторінки манги тепер доступні у повній галереї без обмежень.

---

## 💡 Поради для стабільної роботи
*   **Енергозбереження**: Android намагається закривати фонові процеси для збереження батареї. Скрипт автоматично активує функцію `termux-wake-lock`, яка повідомляє операційній системі не вивантажувати Termux з пам'яті під час перекладу.
*   **Перезавантаження**: Якщо ви перезавантажили телефон, просто один раз відкрийте додаток Termux — автостарт самостійно оновить роботу сервера перекладу `llama-server` та веб-інтерфейсу.
