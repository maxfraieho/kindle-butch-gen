# 🦦 Vydra — документація українською

Vydra — інструмент, що перекладає книги українською, генерує аудіокниги та
адаптує манґу. Все працює **локально, на вашому власному Android-пристрої**:
без хмари, без підписок, без збору даних. Створено в Україні під час війни.

## Що вміє Vydra

- **Книги (EPUB/PDF)** — переклад українською локальною LLM (Hy-MT2-7B),
  збірка EPUB та AZW3 для Kindle.
- **Аудіокниги** — синтез українського мовлення (Supertonic TTS) з наголосами
  та природними паузами.
- **Манґа** — розпізнавання тексту, переклад, очищення бульбашок і typeset
  українською; експорт у CBZ та AZW3; ручний редактор бульбашок
  (текст + геометрія) прямо в браузері.
- **Живе редагування** — правки тексту/аудіо/манґи можна вносити, поки
  генерація ще триває; система сама перегенерує зачеплене.

## Принципи

1. **Локальність.** Ваші книги не залишають ваш пристрій. Хмара (Appwrite)
   використовується лише для реєстрації, рефералів і прапорців профілю.
2. **Безкоштовна база.** Переклад, аудіо і манґа — безкоштовні назавжди.
   Преміум — лише додаткові функції якості (див. [бот і підтримка](bot.md)).
3. **Прозора підтримка.** Примітка підтримки в книгах чітко розділяє
   офіційний фонд (Повернись живим) і підтримку розробника — і вимикається
   однією командою.
