# 🦦 Vydra — документація українською

Vydra — інструмент, що перекладає книги українською, генерує нейронні аудіокниги з наголосами, виконує автоматичну верифікацію вимови та адаптує манґу. Усе працює **локально, на вашому власному Android-пристрої**: без хмари, без підписок, без збору даних. Створено в Україні під час війни.

## 🚀 Що вміє Vydra

- **📚 Книги (EPUB/PDF)** — переклад українською локальною LLM (Hy-MT2-7B) з автоматичною збіркою EPUB та AZW3 для Kindle.
- **🎙️ Нейронні аудіокниги (Supertonic 3)** — синтез українського мовлення (Flow Matching, 10 дикторів) з правильними наголосами та паузами. Прискорено через OpenCL / NNAPI.
- **🔍 ASR-верифікація наголосів (Whisper Small INT8)** — автоматичне розпізнавання мовлення через ONNX Runtime для знаходження некоректно вимовлених наголосів.
- **🧠 MQM-оцінка якості перекладу** — автоматична оцінка перекладених абзаців за шкалою Multidimensional Quality Metrics з виявленням смислових викривлень.
- **🤖 Агент-редактор (Gemma 3 4B)** — ШІ-аналіз складних сторінок манґи та тексту моделлю зору. Агент пропонує виправлення у черзі `Pending Edits` з порівнянням «до/після» лише за вашою згодою.
- **🎨 Манґа** — розпізнавання тексту, переклад, очищення бульбашок і typeset українською; експорт у CBZ та AZW3; інтерактивний редактор прямо в браузері.
- **🔄 Живе редагування** — правки тексту, наголосів чи геометрії можна вносити під час генерації; система сама перегенерує зачеплені фрагменти.

## 🛡️ Принципи

1. **Локальність.** Ваші книги не залишають ваш пристрій. Хмара (Appwrite) використовується лише для реєстрації, рефералів і прапорців профілю.
2. **Безкоштовна база.** Переклад, аудіо і манґа — безкоштовні назавжди. Преміум — лише додаткові функції якості (див. [бот і підтримка](bot.md)).
3. **Прозора підтримка.** Примітка підтримки в книгах чітко розділяє офіційний фонд (Повернись живим) і підтримку розробника — і вимикається однією командою.
