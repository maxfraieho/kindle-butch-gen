# Бот, підтримка і преміум

Telegram-бот [@GetVydraBot](https://t.me/GetVydraBot) — реєстрація, реферали, керування примітками підтримки та розширеними можливостями.

---

## 🤖 Команди бота

| Команда | Призначення |
|---|---|
| `/start` | Реєстрація (без паролів/email), показ вашого Telegram ID та реферального посилання |
| `/install` | Покрокова інструкція із розгортання Vydra на Android |
| `/referral` | Ваш реферальний код та статистика запрошених друзів |
| `/no_support_banner` | Перемикач вимкнення приміток підтримки в книгах (миттєво і назавжди) |
| `/premium` | Перевірка стану розширених можливостей та інструкція з їх активації |
| `/menu` | Інтерактивне меню кнопками |

---

## ☕ Примітки підтримки в книгах (Track A / Track B)

У згенерованих книгах зрідка (на паузах між розділами) з'являється коротка примітка:

- **🇺🇦 Track A — захисникам**: посилання на офіційну сторінку фонду «Повернись живим» (публічна звітність).
- **☕ Track B — розробнику**: окреме посилання для підтримки автора інструменту.

**Вимкнення**: команда `/no_support_banner` у боті або тумблер у картці «🦦 Примітки підтримки» на дашборді.

---

## 💎 Преміум-можливості та багатопристроєвий доступ

Базова функціональність (переклад, аудіокниги Supertonic 3, манґа) — **безкоштовна назавжди**.

Преміум відкриває розширені ШІ-модулі підвищеної якості:
1. **🎙️ ASR-верифікація наголосів (Whisper Small INT8)** — автоматичне виявлення помилок вимови в аудіокнигах.
2. **🧠 MQM-оцінка якості перекладу** — глибокий аналіз точності та повноти перекладу.
3. **🤖 Агент-редактор (Gemma 3 4B)** — візуальний аналіз складних сторінок та шрифтів.
4. **🧬 Cast Registry** — реєстр персонажів із фіксацією граматичного роду.
5. **📱 Розширений ліміт пристроїв** — знімає обмеження на безкоштовні 3 пристрої на акаунт.
