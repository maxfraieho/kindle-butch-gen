# Встановлення Vydra на Android

> Найкоротший шлях: надішліть команду `/install` боту
> [@GetVydraBot](https://t.me/GetVydraBot) — він проведе вас цими ж кроками.

## 📋 Вимоги до системи

| Параметр | Мінімум | Рекомендовано |
|---|---|---|
| Архітектура | 64-біт (aarch64) | Snapdragon 8 Gen 1/2/3 |
| Оперативна пам'ять | 6 GB | 10+ GB (модель перекладу займає ~5 GB) |
| Вільне місце | 15 GB | 25+ GB (переклад 4.4 GB + контейнер ~8 GB + преміум-моделі) |
| GPU | не обов'язково | Adreno з OpenCL — прискорює переклад та аудіосплав |
| Мережа | Wi-Fi на час встановлення (завантажується ~6 GB) | |

Скрипт розгортання **сам перевіряє всі вимоги** на старті (діагностика PASS/WARN/FAIL) і зрозуміло пояснює, чого бракує.

## 📥 Крок 1. Termux — тільки з F-Droid

Встановіть **Termux з F-Droid**: https://f-droid.org/packages/com.termux/

⚠️ Версія з Google Play Market мертва роками — з нею нічого не працюватиме.

---

## 🔋 Крок 2. Вимкніть оптимізацію батареї для Termux

**Це найчастіша причина невдалих встановлень.** `termux-wake-lock` (який скрипт вмикає сам) блокує лише засинання CPU — окремий механізм Android (Battery Optimization / Adaptive Battery) все одно може вбити Termux у фоні.

Налаштування телефона → Застосунки → **Termux** → Батарея → **Без обмежень** (або Оптимізація батареї → Усі застосунки → Termux → **Не оптимізувати**).

---

## 👻 Крок 2.5. Фікс "Phantom Process Killer" (Android 12+)

Якщо переклад чи генерація аудіокниги стабільно зникає кожні ~20–40 хвилин роботи (навіть із вимкненою оптимізацією батареї), увімкнено внутрішній обмежувач Android.

Виконайте у Termux:

```bash
bash ~/kindle-butch-gen/bin/fix_phantom_process_killer.sh
```

Скрипт покроково проведе через увімкнення "Бездротового налагодження" (Developer Options) і зніме обмеження Android.

## 🚀 Крок 3. Termux:Boot (автостарт після перезавантаження)

Встановіть **Termux:Boot** з F-Droid і **відкрийте його один раз**: https://f-droid.org/packages/com.termux.boot/

Детальніше про налаштування автостарту: [docs/deployment/termux-boot-setup.md](../deployment/termux-boot-setup.md).

## ⚙️ Крок 4. Одна команда розгортання

Відкрийте Termux і вставте:

```bash
bash <(curl -fsSL https://raw.githubusercontent.com/maxfraieho/kindle-butch-gen/master/deploy.sh) -a
```

Скрипт виконає діагностику, налаштує Ubuntu 24.04 LTS у `proot-distro`, зкомпілює `llama.cpp` з прискоренням OpenCL, встановить `Supertonic 3 TTS` та підготує веб-дашборд.

## 📦 Розширені моделі ШІ (Преміум)

При активації розширених функцій у налаштуваннях книги (`⚙️ Налаштування книги`) веб-інтерфейс перевірить наявність локальних моделей і запропонує їх фонове дозавантаження:

| Модель | Призначення | Розмір |
|---|---|---|
| **Whisper Small INT8** | ASR-верифікація наголосів у аудіокнигах | ~245 МБ |
| **Gemma 3 4B Vision** | Агент-редактор манґи та складного тексту | ~3.3 ГБ (GGUF + mmproj) |

Ручне завантаження моделей через CLI:
```bash
./bin/download_premium_models.sh --target all
```
