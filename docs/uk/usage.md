# Користування Vydra (UI 2.0)

Веб-інтерфейс Vydra працює на порту `5000` (`http://localhost:5000` з телефона або `https://kindle.exodus.pp.ua` у мережі).

---

## 🎨 Головний екран (Dashboard UI 2.0)

- **➕ Add Book** — додати книгу: EPUB/PDF (текст) або архів/теку зображень (манґа).
- **⚙️ Налаштування книги** — відкриває блок керування конкретною книгою, зокрема преміум-функціями (ASR, MQM, Агент-редактор).
- **🤖 Models Manager** — стан локального сервера перекладу `llama.cpp` та керування ним (Start / Stop / Restart).
- **🔄 Update** — оновлення Vydra однією кнопкою (безпечно чекає завершення активних генерацій).
- **⚙ Save Location** — вибір папки для експорту готових файлів (`.epub`, `.azw3`, `.mp3`, `.cbz`).
- **🦦 Примітки підтримки** — стан інформаційного банера та прив'язка пристроїв.

---

## 👑 Преміум-перемикачі у Налаштуваннях книги

При натисканні кнопки **⚙️ Налаштування книги** у картці книги відкривається панель керування UI 2.0 з блоком розширених можливостей:

> [!TIP]
> **Блок «👑 Преміум-можливості (UI 2.0)»**
> 
> - **🎙️ ASR-верифікація наголосів (Whisper)** (`enable_asr_verify`) — запускає фонову перевірку вимови синтезованого аудіо через модель Whisper Small INT8 (~245 МБ). Знайдені відхилення відправляюся у чергу корекції наголосів.
> - **🧠 MQM-оцінка якості перекладу** (`enable_mqm_review`) — запускає аналіз перекладу моделлю-рецензентом за 10-бальною шкалою Multidimensional Quality Metrics для виявлення смислових пропусків чи викривлень.
> - **🤖 Агент-редактор (Gemma 3 4B)** (`enable_agent_editor`) — активує візуальний аналіз складних сторінок манґи та форматування.

При першому ввімкненні перемикача система автоматично перевіряє наявність потрібних нейромережевих моделей на диску. Якщо модель відсутня, виводиться модальне вікно підтвердження на її завантаження по Wi-Fi.

## 📊 Visual Visualizer та Редагування (`/view/<книга>`)

Натискання кнопки перегляду на картці книги відкриває Quality Visualizer:

1. **Порівняння сторінок**:
   - `Original` — оригінальна сторінка / абзац.
   - `Cleaned` — очищений від шуму чи бабл-тексту варіант.
   - `Translated` — готовий переклад українською мовою.

2. **Черга «Pending Edits»**:
   - Правки, запропоновані **Агентом-редактором (Gemma 3 4B)** або внесені вручну.
   - Надає порівняння «як є / як буде».
   - **Жодні зміни не застосовуються без вашого підтвердження**.

3. **Інтерактивний редактор манґи**:
   - Клік по бульбашці відкриває вікно редагування тексту та геометрії.
   - Зміна тексту або розмірів бульбашки викликає фонову перегенерацію лише зачепленої сторінки з найвищим пріоритетом.

## 🎙️ Управління аудіокнигами (Supertonic 3)

- Підтримує 10 високоякісних дикторських голосів (`0` – `9`).
- Вбудований слайдер швидкості мовлення (`0.8x` – `1.5x`).
- Функція миттєвого прослуховування аудіопрев'ю перед повною генерацією книги.
- Автоматичний кєш наголосів (`stress_cache_uk.json`) та генерації аудіо-чанків (`tts_cache_supertonic-3-tts-int8.json`).

## 📱 Робота з кількома пристроями (до 3 пристроїв безкоштовно)

- До одного акаунта Telegram можна прив'язати **до 3 пристроїв безкоштовно**.
- Усі пристрої працюють повністю незалежно й автономно.
- Для прив'язки нового пристрою відкрийте `/start` у [@GetVydraBot](https://t.me/GetVydraBot), скопіюйте ваш Telegram ID і вставте його в картку підтримки нового пристрою.
