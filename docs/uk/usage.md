# Користування Vydra

Веб-інтерфейс: `http://localhost:5000` (з телефона) — усе керується звідти.

## Головний екран (Dashboard)

- **➕ Add Book** — додати книгу: EPUB/PDF (текст) або тека/архів зображень
  (манґа). Вкажіть назву, автора, мову оригіналу.
- **🤖 Models Manager** — стан і перезапуск сервера перекладу.
- **🔄 Update** — оновлення Vydra однією кнопкою: сервіс сам стягне свіжий
  код і перезапуститься (під час активної генерації оновлення чемно
  відмовиться — дочекайтесь завершення).
- **⚙ Save Location** — куди складати готові файли (пам'ять телефона
  доступна після `termux-setup-storage`).
- Картка **🦦 Примітки підтримки** — стан банера підтримки та керування ним
  (детальніше — у [боті й підтримці](bot.md)).

## Картка книги

Прогрес по стадіях (переклад → наголоси → аудіо / OCR → очищення → typeset),
кнопки **Run / Stop / Re-run / Delete**, журнал виконання просто в картці.
Готові файли — кнопки завантаження EPUB/AZW3/CBZ/аудіо там само та в
**📥 Downloads Archive**.

Генерацію можна безпечно переривати: повторний Run продовжує з місця
зупинки (посторінкова/почанкова resumability). Якщо Termux впав посеред
конверсії — після перезапуску вона **відновиться автоматично**.

## Перегляд якості та редагування (`/view/<книга>`)

Кнопка перегляду на картці книги відкриває Quality Visualizer:

- **Перемикач Original | Cleaned | Translated** — порівняння постранично.
- **Auto-flagged** — сторінки, які система сама позначила як підозрілі
  (кольори: 🔴 проблема, 🟡 увага, 🟢 ок) — з причиною кожного прапорця.
- **Pending Edits** — ваші правки в черзі з diff-переглядом до/після.

### Редагування манґи

Клік по бульбашці відкриває редактор:

- **Текст** — виправте переклад, збережіть; сторінка перегенерується у фоні
  з пріоритетом (випереджає чергу нових сторінок).
- **Геометрія** — тягніть бульбашку за тіло (перемістити) або за кутові
  маркери (розмір); система перевіряє перекриття сусідніх бульбашок.
- Правки **переживають** подальші регенерації сторінки: затверджене не
  губиться і не перезаписується автоматикою.

### Редагування тексту/аудіо

У текстових книгах правка абзацу ставить у чергу перегенерацію лише
зачеплених аудіо-чанків (не всієї книги). Все це можна робити, **поки
основна генерація ще триває** — черга розрулює пріоритети сама.

## Куди що зберігається

| Що | Де |
|---|---|
| Книги і весь робочий стан | `~/kindle-butch-gen/books/<назва>/` |
| Готові файли для читалок | тека з ⚙ Save Location + Downloads Archive |
| Журнали сервісів | `~/kbg-flask.log`, `~/llama-translation-server.log` |
| Журнал оновлень | `~/kbg-update.log` |
