1. Головний екран

Головний екран: картка підтримки, кнопки дій і список книг із прогресом стадій

  • ➕ Add Book — додати книгу: EPUB/PDF (текст) або теку/архів зображень (манґа).
  • 🤖 Models Manager — стан сервера перекладу і його перезапуск.
  • 🔄 Update — оновлення Vydra однією кнопкою: сервіс сам стягне свіжий код і перезапуститься.
  • ⚙ Save Location — тека для готових файлів (потрібен дозвіл termux-setup-storage).
  • 🦦 Примітки підтримки — стан банера підтримки; керування — в Telegram-боті або локальним перемикачем.

Картка книги

Смуга прогресу і чипи стадій (OCR → Trans → Stress → TTS для тексту; OCR → очищення → typeset для манґи), кнопки Run / Stop / Re-run / Delete і журнал виконання просто в картці.

Генерацію можна безпечно переривати — повторний Run продовжує з місця зупинки. Якщо Termux впав посеред конверсії, після перезапуску вона відновиться автоматично.

2. Перегляд якості та редагування

Quality Visualizer: перекладена сторінка манґи, перемикач Original | Cleaned | Translated, навігація і стрибок на сторінку

  • Вкладка Viewer — посторінкове порівняння: оригінал, очищені бульбашки, перекладений typeset.
  • Вкладка Pending Edits — ваші правки в черзі з diff-переглядом до/після.
  • Auto-flagged — сторінки, які система позначила сама: 🔴 проблема, 🟡 увага, 🟢 ок — з причиною кожного прапорця.

Редагування манґи

  • Клік по бульбашці → редактор: виправте текст перекладу — сторінка перегенерується у фоні з пріоритетом.
  • Геометрія: тягніть бульбашку за тіло (перемістити) або за кутові маркери (розмір); перекриття сусідніх бульбашок перевіряється автоматично.
  • Затверджені правки переживають подальші регенерації сторінки — автоматика їх не перезапише.
Усе це можна робити, поки основна генерація ще триває — черга сама дає пріоритет вашим правкам.

3. Готові файли

Downloads Archive: всі згенеровані EPUB / AZW3 / CBZ / аудіо в одному місці

Файли також доступні кнопками завантаження на картці книги та в теці з ⚙ Save Location — звідти їх зручно кидати на Kindle чи в читалку.

4. Бот, підтримка, преміум

  • @GetVydraBot: /start — реєстрація, /install — встановлення на новий пристрій, /referral — запрошення (+пріоритет черги обом), /no_support_banner — вимкнути примітки підтримки одним кроком, /premium — преміум-функції.
  • Примітка підтримки в книгах чітко розділяє 🇺🇦 офіційний фонд (Повернись живим, публічна звітність) і ☕ підтримку розробника — і вимикається назавжди однією командою.
  • Преміум (Cast Registry — правильний граматичний рід персонажів; Vision-QA — додаткова візуальна перевірка манґи) відкривається після донату. Базова генерація — безкоштовна назавжди.

5. Якщо щось пішло не так

  • Сервіс не відповідає → відкрийте Termux (автостарт підніме все сам).
  • "Сховище Android недоступне" → виконайте termux-setup-storage.
  • Довгі операції → тримайте екран увімкненим: Android вбиває важкі фонові процеси.
  • Повна документація: docs/uk на GitHub.
