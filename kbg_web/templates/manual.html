1. Додавання книги

➕ Add Book: файл або шлях, метадані, мови, перемикач манґи

  1. Файл: завантажте PDF / EPUB / TXT / MD — або вкажіть шлях до файлу, що вже лежить на телефоні.
  2. Slug — коротке технічне ім'я (латиниця, без пробілів), Title та Authors — для EPUB метадані підтягнуться автоматично.
  3. Source Language — мова оригіналу (для EPUB є автовизначення), Target Language — мова перекладу й озвучення (українська).
  4. Is Manga / Comic — увімкніть для манґи/коміксів: замість текстового пайплайна запуститься розпізнавання і typeset зображень (джерело — тека або архів зображень, CBZ/ZIP/PDF).

2. Картка книги: стадії та керування (UI 2.0)

Оновлений інтерфейс Дашборду: лінійний прогрес-бар, великі кнопки першого плану та меню ⚙️ Налаштування ▾

Верхнє меню «⚙️ Налаштування ▾»

Для збереження чистоти інтерфейсу всі другорядні технічні кнопки об'єднані в єдиний дріпдаун у шапці:

  • 🤖 Моделі — перемикач GGUF-моделей та контроль сервера перекладу.
  • 📁 Куди зберігати — вибір вихідної теки для збереження книг і аудіо.
  • 🔑 Пароль — зміна паролю веб-доступу Vydra.
  • ⚙️ Загальні налаштування — налаштування за замовчуванням.
  • 🔄 Оновити Vydra — оновлення системи в 1 клік з інтерактивними Toast-сповіщеннями у правому кутку.

Головні кнопки першого плану (Direct Action)

На кожній картці найважливіші дії завжди залишаються великими та доступними в один клік:

  • 📖 Читати / Редагувати — швидкий перехід у переглядач і редактор перекладу.
  • ⬇️ Завантажити — прямо з картки (для готових AZW3 / EPUB / CBZ / MP3).
  • ▶️ Почати переклад — запуск конвеєра перекладу.
  • ⏸ Зупинити переклад — безпечне зупинення поточного процесу.

Меню контекстних дій (Kebab Menu ⋮)

У правому верхньому кутку кожної картки знаходиться меню ⋮ для технічних та рідкісних дій:

  • ⚙️ Налаштування книги — вибір голосу TTS, роздільності для манґи, мовних параметрів.
  • 🔄 Перекласти заново — повторний повний прогін книги з нуля.
  • 🧾 Технічний журнал — перегляд живого логу конверсії.
  • 🗑️ Видалити книгу — видалення книги та її матеріалів.

Мінімалістичний прогрес-бар

Замість громіздких схем відображається 1 чистий лінійний прогрес-бар із назвою поточного етапу ("Перекладаю (45%)" або "Готово ✅ (100%)").

Генерацію можна безпечно переривати: повторний запуск продовжує з місця зупинки. Інтерактивні Toast-сповіщення у правому кутку відразу повідомляють про статус оновлення та події системи.

3. Models Manager — сервер перекладу

Стан llama-сервера (порт 8081), завантажена модель, Start/Stop

Тут видно, чи запущений сервер перекладу і яка GGUF-модель завантажена. Start/Stop Server — ручне керування (звичайно не потрібне: сервер стартує автоматично). Випадачка Translation Model дозволяє підключити іншу GGUF-модель з ~/models/ — Save Configuration і перезапуск.

4. Текстові книги: читання, мобільні таби та редагування

Десктопний режим перегляду та редагування книги з пофрагментним порівнянням

Мобільний режим (< 768px): таби «Оригінал | Переклад» розгортають текст на 100% ширини

Кнопка 📖 Читати / Редагувати на картці відкриває повнофункціональний переглядач. На мобільних пристроях (< 768px) автоматично вмикаються мобільні таби «Оригінал | Переклад», щоб текст був максимально чітким і зручним для читання.

Вкладка «Paragraphs & Audio» — робота з кожним фрагментом

🖼 Пофрагментне редагування: кожен абзац показано парою «оригінал → переклад» з кнопкою Edit і аудіо-прев'ю.

Кожен фрагмент: оригінал | переклад з наголосами (+), плеєр сегмента, форма Edit із двома полями

  • Два стовпчики: ORIGINAL і TRANSLATED & ACCENTS — переклад із проставленими наголосами (знак + після наголошеного складу керує вимовою TTS).
  • 🎧 Listen Segment Preview — прослухати озвучення саме цього фрагмента.
  • Статуси: Audio Synthesized / No Audio.
  • ✏️ Edit відкриває два поля: Translated text (виправити переклад) і Stress marks (виправити наголоси) → Save — фрагмент стане в чергу пере-озвучення з пріоритетом, решта книги не чіпається.
  • Фільтри: Show All | With Audio Only | Without Audio Only | Pending Edits (ваші правки в черзі та їх статус).
  • Пагінація внизу/вгорі: сторінки по ~30 фрагментів, стрибок на номер — поле # + Go.

Вкладка «Full Page Viewer» — читання по розділах

🖼 Full Page Viewer: перемикач Original/Translated показує цілу сторінку до і після перекладу.

Розділ цілком: вибір файла-розділу, перемикач Original | Translated, навігація між розділами

Зручно вичитувати переклад у контексті: обираєте розділ (випадачка Select Page/File), перемикаєте Original ↔ Translated, гортаєте «Попередній/Наступний розділ».

Усі правки можна вносити, поки основна генерація ще триває — черга сама дає пріоритет вашим виправленням (вони озвучуються першими).

5. Манґа: перегляд і редагування бульбашок

🖼 Переглядач манґи: сторінки у станах Original / Cleaned / Translated з навігацією.

Перекладена сторінка манґи: перемикач Original | Cleaned | Translated, навігація, стрибок на сторінку

  • Viewer — посторінкове порівняння: оригінал → очищені бульбашки → перекладений typeset. Статус сторінки (PROCESSED) поруч із номером.
  • Pending Edits — правки в черзі з різницею до/після.
  • Auto-flagged — сторінки, позначені системою: 🔴 проблема, 🟡 увага, 🟢 ок — з причиною (перекриття рамок, низька впевненість розпізнавання тощо).

Редактор бульбашки

  • Клік по бульбашці → правка тексту перекладу; сторінка перегенерується у фоні з пріоритетом.
  • Геометрія: тягніть бульбашку за тіло (перемістити) чи за кутові маркери (розмір); перекриття з сусідніми бульбашками перевіряється автоматично.
  • Затверджені правки переживають подальші регенерації — автоматика їх не перезаписує.

6. Готові файли

Downloads Archive: пошук, опис кожного файла і цільові пристрої

  • Всі згенеровані файли по книгах: EPUB (Kobo/PocketBook/Apple Books/Android), AZW3 (Amazon Kindle), CBZ (комікс-рідери, повна роздільність), MP3 (аудіокнига, будь-який плеєр).
  • Пошук за назвою/форматом/описом; видалення непотрібного — 🗑 поруч.
  • Ті самі файли доступні чипами на картці книги та в теці ⚙ Save Location — звідти зручно кидати на рідер.

7. Бот, підтримка, розширені можливості

  • @GetVydraBot: /start — реєстрація, /install — встановлення на новий пристрій, /referral — запрошення (+пріоритет черги обом), /no_support_banner — вимкнути примітки підтримки одним кроком, /premium — розширені можливості.
  • Примітка підтримки в книгах чітко розділяє 🇺🇦 офіційний фонд (Повернись живим, публічна звітність) і ☕ підтримку розробника — вимикається назавжди однією командою або перемикачем на головній.
  • Розширені можливості (Cast Registry — правильний граматичний рід персонажів; Агент-редактор — візуальна перевірка й виправлення проблемних сторінок манґи; більше 3 пристроїв на акаунт) відкриваються після донату. Базова генерація — безкоштовна назавжди, на будь-якій кількості пристроїв.

Кілька пристроїв на один акаунт

  • Vydra можна встановити на кілька пристроїв (телефон + планшет тощо) — кожен генерує книги повністю самостійно й локально, без взаємного втручання.
  • Прив'язка нового пристрою: встановіть Vydra так само (deploy.sh), напишіть /start боту тим самим Telegram-акаунтом — бот покаже ваш числовий ID; у картці підтримки нового пристрою з'явиться поле «🔗 Цей пристрій ще не прив'язаний» — вставте туди ID і натисніть Прив'язати.
  • Після прив'язки розширені можливості й статус підтримки спільні для всіх пристроїв. Сповіщення про зупинку конверсії приходять одним дайджестом на акаунт — окремий рядок на кожен пристрій (📱 Телефон: працює, 📱 Планшет: зупинився), не спамом.
  • Безкоштовно — до 3 пристроїв на акаунт. Понад це число пристрій і далі повністю працює (генерація книг ніколи не блокується), але не отримує сповіщень про зупинку — у картці підтримки з'явиться позначка «⚠️ понад безкоштовний ліміт». Розширені можливості знімають обмеження.

Розширені можливості: перші кроки після активації

  • 👑 Управління преміум-функціями у Налаштуваннях книги (UI 2.0) — відкрийте «⚙️ Налаштування книги» у картці книги на дашборді. Спеціальний блок преміум-можливостей дозволяє прозоро вмикати та вимикати розширені нейромережеві модулі для конкретної книги.
  • 🎙️ ASR-верифікація наголосів (Whisper) — для аудіокниг. Автоматично розпізнає синтезоване аудіо через локальну нейромережу Whisper Small INT8 (~245 МБ) для виявлення неправильних наголосів і направлення їх у чергу верифікації наголосів.
  • 🧠 MQM-оцінка якості перекладу — для текстових книг. Модель-рецензент аналізує кожен перекладений абзац за 10-бальною шкалою, знаходить смислові викривлення чи пропуски й позначає їх у звіті якості.
  • 🤖 Агент-редактор (Gemma 3 4B) — візуальний та контекстний ШІ-аналіз складних сторінок. Агент пропонує виправлення (розсунути зону тексту, змінити шрифт) у вкладці «Pending Edits» з візуальним порівнянням «як є / як буде». Без вашого підтвердження нічого не змінюється.
  • 📦 Автоматична перевірка моделей та діалог згоди — при першому ввімкненні перемикачів ASR або Агента-редактора система перевіряє наявність нейромережевих моделей на диску. Якщо моделі відсутні, показується підтвердження на дозавантаження по Wi-Fi.
  • ⚠️ Охолодження — vision-аналіз і переклад сильно гріють телефон. Радимо активне охолодження (магнітний кулер) під час роботи агента і перекладу — без нього Android може зупиняти важкі задачі, а телефон троттлитиме.
  • 🧬 Cast Registry — у переглядачі книги вкладка «Cast & Context»: «Сканувати персонажів (NER)» знайде імена автоматично, далі підтвердіть рід (статус verified) — і переклад відмінюватиме правильно («що зробила Мірея», а не «зробив»).

8. Якщо щось пішло не так

  • Сервіс не відповідає → відкрийте Termux (автостарт підніме все сам).
  • «Сховище Android недоступне» у виборі теки → виконайте termux-setup-storage і дайте дозвіл.
  • Переклад не йде → Models Manager: сервер має бути Running; якщо ні — Start Server.
  • Довгі операції → тримайте екран увімкненим: Android вбиває важкі фонові процеси.
  • Оновлення Vydra → кнопка 🔄 Update на головній (під час активної генерації оновлення чемно зачекає).
  • Повна документація: docs/uk на GitHub.

9. Правові застереження

Vydra — інструмент для особистого перекладу книг/манґи, якими ви вже законно володієте; не хостить і не поширює чужий контент; отриманий переклад заборонено публічно поширювати без згоди правовласника. Повний текст: правові застереження.
